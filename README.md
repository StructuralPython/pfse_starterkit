# pfse_starterkit
A Python package that installs all required packages for the StructuralPython Python for Structural Engineers (pfse) course.

This installs the following packages:

* streamlit
* numpy
* pandas
* openpyxl
* lxml
* sympy
* shapely
* vtk
* matplotlib
* plotly
* sectionproperties
* concreteproperties
* PyNiteFEA
* scipy
* handcalcs
* forallpeople
* pytest
* ipytest
* rich
* tqdm
* papermodels
* xlwings
* black
* jupyterlab
* IPython
* ipywidgets
* ipykernel
* jupyterlab-katex
* typer
* magicgui[pyside2]
